## Installation

1. Create a virtualenv & activate it:

```
virtualenv env
source env/bin/activate
```

2. Install the package locally:

```
pip install -e .
```


## Tests 

(`pip install -r requirements_dev.txt`)

- Unit / Integration tests: `pytest`
- Type checking: `mypy src`
- Linting: `flake8 src`
